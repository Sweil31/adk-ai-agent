# ADK AI Agent Project

This project is built using Google Agent Development Kit (ADK) and deployed on Cloud Run.

## Features
- AI Agent using Gemini (Vertex AI)
- Google Search integration
- Cloud deployment using Cloud Run

## How to Run
1. Install dependencies
2. Activate environment:
   source .venv/bin/activate
3. Run:
   adk web

## Deployment
python3 deploycloudrun.py
